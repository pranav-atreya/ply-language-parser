# Basic Language Constructs using PLY

## Description
This project implements basic language constructs using Python Lex-Yacc (PLY).

## Features
- Lexical analysis using PLY
- Parsing basic constructs (expressions, statements, etc.)
- Demonstrates compiler design fundamentals

## Tools Used
- Python
- PLY (Python Lex-Yacc)

## How to Run
pip install ply
python ply_project.py
